# Configuration — Infomaniak

> Cette page sera complétée avec les détails de configuration spécifiques à Infomaniak.

## Informations requises

- `api_token` : Token API Infomaniak (généré depuis le Manager Infomaniak)
- `base_url` : URL de base de l'API (ex : `https://api.infomaniak.com`)

## Données synchronisées

- Comptes Infomaniak (infomaniak_accounts)
- Produits par compte (infomaniak_products) : service, nom interne, nom client, date d'expiration, statut trial/gratuit
