# Configuration — NinjaOne

> Cette page sera complétée avec les détails de configuration spécifiques à NinjaOne.

## Informations requises

- `client_id` : ID client OAuth2 NinjaOne
- `client_secret` : Secret client OAuth2 NinjaOne
- `base_url` : URL de base de l'API (ex : `https://eu.ninjarmm.com`)

## Données synchronisées

- Organisations (ninjaone_organizations) : compteurs RMM, NMS, MDM, VMM, Cloud
- Appareils (ninjaone_devices) : nom, groupe, OS, statut en ligne, dernière connexion
